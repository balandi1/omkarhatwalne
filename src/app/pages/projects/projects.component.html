Personal Wallet Blockchain

1. A Blockchain is a type of diary or spreadsheet containing information about transactions.

2. Each transaction generates a hash which is a string of numbers and letters. The hash depends not only on the transaction but on the previous transaction's hash. A small change in a transaction creates a new hash.

3. A wallet is a string of numbers and letters, such as 18c177926650e5550973303. This address will appear in various blocks within the Blockchain as transactions take place.

4. No visible records of who did what transaction with who, only the number of a wallet.
